Les accents accessibles sous LaTeX sont les suivants :
+ \`{a} ou \`a accent grave
+ \'{e} ou \'e accent aigu
+ \^{i} ou \^i accent circonflexe
+ \"{o} ou \"o trema
+ \~{u} ou \~u tilde
+ \={o} ou \=o surligné
+ \.{o} ou \.o point
+ \u{o}
+ \v{o}
+ \H{o} trema hongrois
+ \t{oo}
+ \c{c} cédille
+ \d{o} point en dessous
+ \b{o} sousligné
